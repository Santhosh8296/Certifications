Javascript:

Javascript is a programming language and it is a OBSP (Object Based programming language). it is an interpreted programming language. every Browser has its own interpreter to translate the javascript. the interpreter will translate the elements like form, link, image, layer in DOM(Document Object Model in html page) for doing some actions on that element we need javascript programming langugae.javascrpit is a client side script used in web development to handle client side interaction and validations. it supports reusability, dynmaic memeory allocation and code seperation but it will not aupport inhertance and dynamic polymorphisim. it reduces overhead and complexity as it refers the document element like an array.

Drawback of javascript

it is not safe as the code is exposed to client from browser. javascript have browser incompatability as every browser have it own javascript. javascript can be blocked by browser.

Javascript BOM and DOM

Javascript BOM(Browser Object Model) provides a collection of objects that are used to handle a browser Example: Location, Histroty, Navigation and Window. javascript DOM(Document Object Model) provices a collection of objects that are used to handle a document elements Example: Forms, Imgs, anchor and other elements.

BOM (Browser Object Model)

  1. Window
    it provides a set of propertires and methods that are used to handle the browser window like open,close,print,..etc.
  2. History
    it provides properties and menthods that are used to handle the client browser history.
  3. Location
    it proivides the properties and methods that are used to access the client location details like IP address, Protocaol URLS,...etc.
  4. Navigator
    it provides set of properties and methods that are used to access the client browser details like browser name, version name, plugins supported,...etc.
  5. Document
    it provides the properties and menthods that are used to access and manpulate the document elements like forms, links, images,...etc.

Integrating Javascript into webpage:

HTML is used to present the DOM. Javascript is used to manpulate the DOM. Javascript is integrated into a webpage by using any one of the following three methods.
  1. Inline
